# Como instalar como aplicativo real

1. Crie um repositório novo no GitHub, por exemplo `OFICIN-IA-CHECKLIST`.
2. Envie todos os arquivos deste ZIP na raiz do repositório.
3. Entre em **Actions**.
4. Rode **Build Checklist APK**.
5. Baixe o artefato **OFICIN-IA-CHECKLIST-V15-1-APK**.
6. Instale o `app-debug.apk` no celular Android.

Resultado esperado:

- aparece um ícone próprio chamado **Checklist Inteligente**;
- ao abrir o app, abre o Checklist;
- não abre o SaaS;
- usa o mesmo Firebase do SaaS.
